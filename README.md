# Smart Pincode Finder & Post Office Explorer

##  Project Overview
This project is a web application that allows users to search for Indian post offices using a pincode. It provides detailed information about post offices and allows filtering and sorting of results.

##  API Used
India Post Pincode API  
Example: https://api.postalpincode.in/pincode/{pincode}

##  Features
-  Search post offices by pincode
-  Display all post office details
-  Filter by:
- Delivery Status (Delivery / Non-Delivery)
- Branch Type (Head Office / Sub Office)
-  Sort results by name or district
-  Fast and responsive UI

##  Technologies Used
- HTML
- CSS
- JavaScript (Fetch API)

##  How to Run the Project
1. Clone the repository
2. Open index.html in browser
3. Enter a pincode to search
